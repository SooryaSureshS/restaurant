Website Discount Coupons & Vouchers Odoo Apps

This Odoo apps/module is used to fast and easy way to create website discount vouchers to be applied on webshop order for particular customer.
When new user created system will generate an email to him/her within which a randomly selected discount voucher code will be given so in future he/she can use it to get discount on amount of order.

Features

Configuration Apply for All Gift Coupons

Apply configuration to coupon Date, coupons Amount, Available coupons, coupon Type, Enable/Disable coupon by single configuration.

Create Gift Product & Coupon

User can configure specific gift voucher Date and Amount, Also you can print the gift coupon.

All List Of Created Gift Coupons

User can see all created gift coupons in back-end

Automatically Send Mail

When a new user is created then system will send email for discount voucher code which he/she can use if for discount.

Apply Gift Coupon Validation

Validation apply on wrong Coupon Code, Expiry Coupon.

Apply Discount By Unique Coupon code

User can easy to print gift coupon voucher.

Provide Enable/Disable Gift Functionality on Website

User can easy enable and disable gift coupon voucher on website and back-end.

Order Details with Deducted Gift Coupon Discount

User can see sale order details deducted with gift coupon voucher discount in back-end.
Demo Request Video Tutorial

Coupon Product Configuration

User have to enable "Coupon Product" checkbox on product, Only service type product can be configured as coupon product.

All Gift Coupons

User can see all created gift coupons under "All Gift Coupons" menu.

User can create multiple gift coupon and apply different type of discounts, based on coupon product.

User can print Gift Voucher Coupon in PDF format.

Website Voucher Configuration

On website shopping cart view you can activate the voucher by clicking on customize tab and apply voucher code.

Apply Discount Voucher

User have to enter created coupon voucher code here it check the that the user who is logged in, product and product category listed in the voucher and based on the discount percentage given in the coupon voucher that amount will be deducted from total amount.

If voucher is successfully applied, product will added in the cart.

User can see generated sale order with coupons.

Sale order View

Here you can see applied discount voucher code will be shown in the sale order with their amount.

User can see sales order from website.

Discount Coupons History

User can see discount coupon history.

Validation

If this Voucher is not allow for this customer then raise the warning.


If you have entered voucher code is wrong then raise will be error.


We can not use any discount voucher if there discount voucher is not created. It will be raise Error.


System will raise error When voucher is expired and user want to use it.



Can not use coupon because you reached the maximum limit of usage.


Allready applied coupon warning.
